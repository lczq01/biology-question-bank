# 考试卷管理页面创建与集成

## Core Features

- 考试卷管理页面创建

- 考试题排序功能

- 分值设置功能

- 题目删除功能

- 与考试创建页面双向跳转

- 总分计算功能

创建专门的考试卷管理页面，支持考试题排序、分值设置、删除等核心功能，实现与考试创建页面的双向跳转集成，提供完整的试卷配置管理体验
