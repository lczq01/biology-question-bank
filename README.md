# 高中生物题库系统

## 项目概述
专为高中生物学科设计的在线题库管理和考试系统，支持管理员和学生两种角色。

## 技术栈
- **前端**: React + TypeScript + Tailwind CSS
- **后端**: Node.js + Express.js + TypeScript  
- **数据库**: MongoDB + Mongoose
- **文件存储**: 本地文件系统

## 项目结构
```
biology-question-bank/
├── frontend/          # React前端应用
├── backend/           # Node.js后端API
├── docs/             # 项目文档
├── README.md         # 项目说明
└── .gitignore        # Git忽略文件
```
